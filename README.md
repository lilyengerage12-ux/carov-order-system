# Carov Flow · 卡洛维茵全屋定制协同系统

卡洛维茵全屋定制的订单、部门协同和财务管理应用第一版。

## 已实现

- 经营驾驶舱：合同额、回款、进行中订单、毛利润
- 意向客户：客户列表、搜索、新增客户
- 订单中心：全流程进度、部门筛选、新建订单
- 财务管理：定金、下单款、尾款；货款定金、尾款、材料款
- 部门协同：信息、销售、设计、拆单、生产、配送、安装、美容修复、售后
- 响应式界面：支持电脑和手机浏览

## 查看

直接打开 `index.html`，或在仓库 Settings → Pages 中启用 GitHub Pages。

> 当前版本为可交互的产品原型。正式上线版本将继续接入账号权限、数据库、审批、附件、消息通知和审计日志。
